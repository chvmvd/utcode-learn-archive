JavaScriptの記述

これまでに学習してきたHTMLやCSSは、プログラミング言語というよりはマークアップ言語です。これに対し、JavaScriptはブラウザ上で動作する唯一のプログラミング言語です。張り切って学習していきましょう。

HTMLファイル中にJavaScriptを記述する#

<body>
<script>
document.write('Hello World!');
</script>
</body>

JavaScriptを記述するには、scriptタグを使用します。document.write('Hello World!');は、ブラウザ上にHello World!と表示させるプログラムです。

外部ファイル中のJavaScriptを利用する#

CSSと同じように、JavaScriptも外部ファイルに切り出すことができます。JavaScriptを外部ファイルに記述する場合は、scriptタグにsrc属性を指定します。

index.html
<script src="script.js"></script>
script.js
document.write('Hello World!');

scriptタグにsrc属性を指定する場合、中身は必要ありませんが、閉じタグを省略することは許されていません。
